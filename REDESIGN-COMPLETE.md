## 📱 RESPONSIVE DESIGN

**Mobile Breakpoints:**
- Desktop: 1024px+
- Tablet: 768px - 1024px
- Mobile: < 768px

**Mobile Optimizations:**
- All grids collapse to 1-column on mobile
- Process steps stack vertically with rotated arrows
- Hero text scales with clamp() functions
- Padding reduces on mobile (2rem → 1rem)
- Featured blog switches to 1-column layout
- Navigation becomes vertical menu
- Button groups stack vertically

## 🎯 BRAND CONSISTENCY

**Typography Hierarchy:**
- H1: clamp(2.5rem, 5vw, 4rem) - Hero titles
- H2: clamp(2rem, 4vw, 3rem) - Section titles
- H3: clamp(1.5rem, 3vw, 2rem) - Card titles
- Body: 1rem base, 1.125rem for large text
- Line heights: 1.2 (headings), 1.6-1.8 (body)

**Color Usage:**
- Primary CTA buttons: Bright blue gradient
- Secondary buttons: Transparent with white border
- Card accents: Bright blue borders/highlights
- Stats/numbers: Gradient blue text effect
- Success: Green (#00AA00, #51CF66)
- Error/Before: Red (#CC0000, #FF6B6B)

**Spacing Rhythm:**
- Section padding: 4-6rem vertical
- Card padding: 2-3rem
- Grid gaps: 1.5-2rem
- Element margins: 1-2rem

## 🎨 CUSTOMIZATION GUIDE

**To change colors:**
Edit CSS variables in `themes/lbs/static/css/style.css`:
```css
:root {
  --navy: #0A1628;
  --bright-blue: #0066FF;
  /* etc. */
}
```

**To add a new section:**
Create a new partial in `themes/lbs/layouts/partials/components/`

**To update a page:**
Edit the section template in `themes/lbs/layouts/section/[page].html`

**To add blog posts:**
Create markdown files in `content/blog/`

**To modify navigation:**
Update `hugo.toml` menu configuration
