## 📋 DNS RECORDS TO ADD

```
Type: A | Name: @ | Value: 185.199.108.153 | TTL: 600
Type: A | Name: @ | Value: 185.199.109.153 | TTL: 600
Type: A | Name: @ | Value: 185.199.110.153 | TTL: 600
Type: A | Name: @ | Value: 185.199.111.153 | TTL: 600
```

### **Add This CNAME Record:**

```
Type: CNAME | Name: www | Value: wilsontechconsulting-png.github.io | TTL: 1 Hour
```

---

## 🔧 STEP-BY-STEP

1. **Delete Existing Records** (if any exist for @ or www)
   - Look for A records with Name = "@"
   - Look for CNAME records with Name = "www"
   - Delete them

2. **Add 4 A Records:**
   - Click "Add"
   - Type: A
   - Name: @
   - Value: 185.199.108.153
   - Click "Save"
   - Repeat for the other 3 IPs (.109, .110, .111)

3. **Add CNAME for www:**
   - Click "Add"
   - Type: CNAME
   - Name: www
   - Value: wilsontechconsulting-png.github.io
   - Click "Save"

4. **Save All Changes**
